fetchData#

â–¸ fetchData(keyOrKeys?): Promise<Object>

Since getData exclusively returns data that is stored on the blockchain, fetchData comes in handy. Additionally to the data from the blockchain, fetchData also returns data from IPFS or HTTP(s) endpoints stored as JSONURL or ASSETURL.

info

To ensure data authenticity fetchData compares the hash of the fetched JSON with the hash stored on the blockchain.

getData#

â–¸ getData(keysOrKeys?): any

Gets decoded data for one, many or all keys of the specified ERC725 smart-contract. When omitting the keyOrKeys parameter, it will get all the keys (as per ERC725JSONSchema definition).

caution

Data returned by this function does not contain external data of JSONURL or ASSETURL schema elements.

If you would like to receive everything in one go, you can use fetchData for that.

Parameters#

NameType
keysOrKeys?string | string[]

Returns#
